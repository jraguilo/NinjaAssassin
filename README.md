Ninja Assassin
========
A ninja rougelike game developed in HTML5 and JQuery.

Controls:
w = move up
s = move down
a = move left
d = move right
arrow keys = shoot 
r = continue game
m = mute/unmute music

Mobile Controls:
swipe = move
2 finger swipe = shoot
tap = enable shooting mode
double tap = mute/unmute music
